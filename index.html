Instructions to be followed.

1. Enter the number of forms you want to get filled in the "Enter Number of Forms" field.

2. Select the same number of forms you mentioned above by clicking the "Select PDF files" button.

3. Download the Excel Sheet and fill the details in the same format.

4. Upload that Excel file by clicking the "Select Excel" button.

5. Click the "Convert" button to initiate the conversion process.

6. Wait for the file to process, once finished it will automatically be downloaded to your system.
